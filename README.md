# Análise de Dados - Supermercado

Projeto de portfólio focado em análise exploratória de dados e visualização.
1. Contexto do Negócio
2. Saúde Financeira Geral
3. Onde o lucro está sendo gerado
4. Onde o prejuízo está acontecendo
5. Descontos: vilão ou aliado?
6. Diferenças regionais
7. Recomendações Executivas
